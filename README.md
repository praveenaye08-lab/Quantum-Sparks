# 🧬 IBM Granite Drug Analyzer — Enhanced

> AI-powered drug interaction checker and alternative medicine recommender, built on **IBM Granite 3.0 2B Instruct** and deployable via Google Colab.

---

## 📌 Overview

This project provides two deployable interfaces for clinical drug analysis:

| Interface | File | Description |
|-----------|------|-------------|
| Streamlit Web App | `enhanced_granite_drug_app.py` (auto-generated) | Full UI with tabs, export, quick interaction checker |
| FastAPI Backend | Cell 2 in notebook | REST API for programmatic access |

Both are designed to run inside **Google Colab** and be exposed publicly via **ngrok**.

---

## 🚀 Features

- 🤖 **IBM Granite 3.0 2B** — local LLM inference for clinical recommendations
- 💊 **8+ Drug Database** — aspirin, warfarin, metformin, lisinopril, ibuprofen, acetaminophen, amlodipine, omeprazole
- 🔄 **Alternative Recommendations** — same-class, different-class, and natural alternatives
- 👴 **Age-Specific Dosing** — pediatric, adult, and elderly dosing guidelines
- 🏥 **Condition-Based Suggestions** — diabetes, hypertension, heart disease, kidney/liver disease
- 💰 **Cost Comparison** — relative cost tiers for each medication
- ⚠️ **Drug Interaction Checker** — pairwise and multi-drug interaction alerts
- 📋 **Clinical Report Export** — download as `.txt` or `.json`
- 🌿 **Natural Alternatives** — complementary options with safety notes

## ▶️ How to Run in Google Colab

### Step 1 — Open the notebook in Colab

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/YOUR_USERNAME/YOUR_REPO/blob/main/granite_drug_analyzer.ipynb)

> Replace `YOUR_USERNAME` and `YOUR_REPO` with your actual GitHub details.

### Step 2 — Run Cell 1 (Streamlit App)

This cell:
- Installs all required packages (`torch`, `transformers`, `streamlit`, `pyngrok`, etc.)
- Writes `enhanced_granite_drug_app.py`
- Launches the Streamlit server on port `8501`
- Creates a public ngrok URL

```
🌐 PUBLIC URL: https://xxxx.ngrok.io  ← Click this to open the app
```

### Step 3 — (Optional) Run Cell 2 (FastAPI Backend)

Starts a FastAPI server on port `8000` with:
- `GET /health` — health check
- `POST /analyze` — accepts drug list, age, and conditions; returns Granite analysis

### Step 4 — (Optional) Run Cell 3 (Cleanup)

Disconnects all existing ngrok tunnels cleanly before restarting.

## 📡 API Reference (FastAPI — Cell 2)

### `POST /analyze`

**Request body:**
```json
{

  "age": 75,
  "medical_conditions": "atrial fibrillation, hypertension"
}
```

```json
{
  "granite_analysis": "IBM Granite's clinical analysis text..."
}
```
